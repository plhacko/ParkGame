For proper map request functionality Mapbox access token is need. 
The access token can be retrieved from https://www.mapbox.com/ after 
creating an account.

The access token then needs to submitted from the top Unity editor panel Mapbox > Setup > Access Token.

For the script MapDisplayer.cs to function Mapbox access token needs to be inserted into its public member variable 
'Access Token' from the Inspector or within code.
